Hi. This is a personal project.

Mostly for learning how to use finance/yfinance tools with python.

This is a simple portfolio analytics tool/library.

Most of the library is in portfolio.py. Main.py simply prints a report to the console.

Handles retursn, betas, alphas, correlation matrix, average correlation, capm required return, volatility.

To use main.py, input your portfolio as a dictionary, with yfinance tickers as keys and weights as values.

All data from yfinance.

'Market' is the Russ3000 for beta, alpha, capm calculations.
